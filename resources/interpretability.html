5.4. Interpretability Resources#

Machine learning interpretability has a fancier alias, called Explainable AI (XAI). You can read the bookMolnar [2022] for a deep dive into ML interpretability.

There are a few tools available, namely:

  • Scikit-learn Inspection [Pedregosa et al., 2011]

  • SHAP (not in the best state currently, but beautiful if it works) [Lundberg and Lee, 2017]

  • Alibi [Klaise et al., 2021]

  • Captum for Pytorch [Kokhlikyan et al., 2020]

  • Deepviz Toolbox [Yosinski et al., 2015]
